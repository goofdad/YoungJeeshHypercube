# Summary

This Is my remix of tech 2cs bowden mount this mount lets you squeeze about 18mm extra out of your Z height on a prusa or hyper cube. Ive included the step file as well so if someone feels the need to make changes go for.

The holes on the side are m3 and spaced 10mm apart so anyone who wants to make their own attachments may do so I have printed this so I know that it works and fits the original parts.
